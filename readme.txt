MaxX Single Property Schema For Houzez Theme adds schema markup for single property pages to improve SEO and structured data presentation.

== Description ==
This plugin generates dynamic schema for single property pages using the Houzez theme, including breadcrumb and product schema.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/maxx-property-schema` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to 'MaxX Schema' under the Settings menu to configure options.

== Frequently Asked Questions ==

= What is schema markup? =
Schema markup is code that you put on your website to help search engines return more informative results for users.

= How does this plugin improve my SEO? =
By adding structured data to your property pages, search engines can better understand the content of your site, which can lead to improved visibility in search results.
